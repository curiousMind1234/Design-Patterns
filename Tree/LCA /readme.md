"I do a post-order DFS. At each node, I recurse into left and right subtrees looking for p or q. If a node equals p or q, I return that node upward as a candidate. Back at each parent, if both left and right recursion returned non-null, this node is where p and q's paths split — so it's the LCA. If only one side returned non-null, I forward that result upward unchanged, since the actual answer was found deeper in that subtree. Time complexity is O(n), space is O(h) for the recursion stack."
